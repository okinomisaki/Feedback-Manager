# Feedback Manager

GitHub課題提出に対するフィードバックを、
チェック形式で効率的に作成するための個人用Webツールです。

※ リリース予定はなく、個人利用を目的としています。

---

## 概要

講師業務の中で、毎回似たフィードバック文章を作成する負担を減らすために作成しました。

チェック項目を選択すると、

- 提出 or 再提出の判定
- フィードバック文章の生成

が自動で行われるツールです。

---

## 主な機能

- チェックリスト形式でのフィードバック作成
- 提出 / 再提出の自動判定
- フィードバック文のリアルタイム生成
- ワンクリックでコピー
- localStorage によるデータ保存

---

## 技術スタック

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
